vue项目
	前端vue工程文件。除杨坤之外，其他人在修改调试代码后不上传打包后的文件，只上传你对应修改了的vue工程项目文件里面的文件。调试好的代码，杨坤统一更新到他的本地统一打包上传。你们自己本地调试时是需要自己打包看效果的，只是打包的文件不上传，只提交修改的源文件。

	每次更新vue工程项目的时候，请注意看package.json文件是否更新过，如果有更新，请在自己的本地用npm工具下载相应的包。
	每次上传源文件代码时，请注意只上传自己修改的需要提交的文件。你自己本地的node_modules文件夹都不需要上传。

	源文件使用：
	1.本地需要nodejs环境，npm包工具，webpack打包工具
	2.拉取代码后到对应的vue项目根目录下，命令行：npm install
	3.npm run dev开发环境下查看效果
	4.npm run build打包源代码
	5.将打包的文件，移动到edu项目中对应的文件中，浏览器查看实际效果
